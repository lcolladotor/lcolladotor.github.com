#Visium_SPG_AD #spatialDLPFC #spatialLIBD Aplicación shiny de spatialLIBD spatialLIBD en Bioconductor
Image credit: bioRxiv

En febrero de 2020, publicamos la primera versión preliminar utilizando la plataforma 10x Genomics Visium para transcriptómica espacial. Para este proyecto, creamos el paquete de Bioconductor spatialLIBD y comenzamos a desarrollar nuevos métodos analíticos para este tipo de datos. Dado que compartimos todos los datos y el código cuando publicamos la preimpresión, cuando la publicación revisada por pares y se hizo pública en febrero de 2021, otros ya habían utilizado nuestro conjunto de datos para desarrollar y mostrar sus métodos. Por ejemplo, BayesSpace preimpresión se publicó en septiembre de 2020, antes a su propia publicación revisada por pares de junio de 2021. Ahora estamos usando BayesSpace en nuestros proyectos. Este es un ejemplo de cómo la ciencia abierta y el acceso abierto aceleran la ciencia.
